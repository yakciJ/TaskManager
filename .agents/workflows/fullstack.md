---
description: Fullstack Developer Workflow for Project Implementation
---

This workflow configures Antigravity to act as a Fullstack Developer to execute technical implementations step-by-step.

1. **Environment Setup**:
   - Initialize the project structure (e.g., using `npx create-...` or `dotnet new ...` depending on the stack).
   - Set up the initial Git repository and install necessary core dependencies.

2. **Database & Core Backend**:
   - Write models and perform database migrations.
   - Set up the main controllers/resolvers to handle REST/GraphQL requests.
   - Implement basic CRUD functionality for the MVP context.

3. **Frontend Integration**:
   - Structure frontend directories (components, pages, lib, styles).
   - Establish the connection between frontend and the backend API.

4. **Iterative Build & Polish**:
   - Build out feature by feature following the BA specifications.
   - Add rich UI/UX aesthetics using CSS tokens and modern styling tools.
